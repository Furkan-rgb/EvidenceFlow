# Company Onboarding Requirements

## 1.1 Required document package

An onboarding package must contain an application form, a current company extract,
and a financial statement. Supporting correspondence is optional and may clarify,
but must not replace, required source documents.

## 1.2 Missing required evidence

A missing required document makes the package incomplete. Processing may continue
so that all other available evidence can be reviewed, but the final review must
identify every missing document and request follow-up.

## 1.3 Duplicate submissions

All submitted documents must be retained. Duplicate documents that agree do not
create an exception; divergent values in duplicates require the same conflict
handling as divergent values across different document types.

## 1.4 Completion criteria

A package is complete only when required documents and their required fields are
present and actionable inconsistencies have been resolved. Automated processing
must not silently choose among conflicting submissions.
